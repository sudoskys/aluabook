# When的用法

------------------

{{< hint info >}}
**From MLUA**
{{< /hint >}}

When就相当于简化过的if，对于一些简单的判断使用when会爽不少

例如下面这个栗子

``````lua
--以前判断要这样写

if 5>1 then

print("OK")

end
``````

在androlua+4.3.3更新后可以这样写

``````lua
when 5>1 print("OK")
``````

以前判断要这样写

``````lua
if 1>5 then

print("emmmm")

else

print("OK")

end
``````

在androlua+4.3.3更新后可以这样写

``````lua
when 1>5 print("emmmm") else print("OK")
``````
