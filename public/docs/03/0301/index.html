    为什么要这么架构工程？ #

    根据安卓标准，我们应该使用合格的工程架构目录。 本文编写人推荐使用的工程目录如下：

    
.
├── layout
│   ├── action
│   └── formal
├── libs
├── res
│   ├── drawable
│   └── source
├── run
│   ├── base.lua
│   └── mods
└── value
    └── base

    工程模板在 前缀+/res/frame.zip
